Interactive Graph Algorithm Visualizer

Build, edit, and visualize graphs in the browser with features including:

  • visual node and edge editing
  • directed and weighted graph support
  • BFS and DFS traversal animation
  • Dijkstra and Bellman-Ford, shortest path visualization
  • Prim’s minimum spanning tree visualization
  • manual step-by-step traversal
  • pan and zoom
  • local saves, import/export, undo, and redo
